## Task 2: How to use TryHackMe

This task just let us know about how to use TryHackMe VM through attackbox or SSH.
If you want to learn everything about attackbox here is the official link
https://help.tryhackme.com/en/articles/6721845-the-attackbox-explained

![Screenshot](/CTF-Writeups/AoC2025/Day%200/Task2-1.png)

Further it we can learn how to start a Machine. You would like to do this step everytime you start a new room. This provides you an ip which you can use access using the VM.

![Screenshot](/CTF-Writeups/AoC2025/Day%200/Task2-2.png)

Next if you dont want you use the inbuilt AttackBox as it lags sometime you can connect through SSH. 
Here is a Detailed Step by Step on how to connect to it
https://help.tryhackme.com/en/articles/6502085-how-to-ssh-into-a-machine

```bash
ssh tryhackme@Machine-IP
```
